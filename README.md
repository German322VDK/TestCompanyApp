# TestCompanyApp



## Тестирование
Проект включает в себя модульные тесты для CommentStore, PostStore и TradStore. Чтобы запустить тесты, используйте GitHub Actions или выполните тесты локально с помощью следующей команды:
 ```bash
  dotnet test
 ```
